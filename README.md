# 🎵 Audio Spectrum Visualizer

ブラウザ上で動作するオーディオスペクトラムビジュアライザーです。音楽ファイルを読み込み、リアルタイムで周波数スペクトラムを可視化します。

![Audio Spectrum Visualizer](https://img.shields.io/badge/version-1.0.0-purple)
![License](https://img.shields.io/badge/license-MIT-blue)

## ✨ 機能

### 🎵 音声再生
- MP3, WAV, OGG, M4A などの音声ファイルに対応
- ドラッグ＆ドロップまたはクリックでファイル選択
- 再生/一時停止、経過時間表示

### 📊 ビジュアライゼーション
4種類の表示スタイルから選択可能：
- **バー**: 縦棒グラフ形式（128本）
- **ウェーブ**: 波形表示
- **サークル**: 円形放射状表示（180本）
- **パーティクル**: 粒子表示（64個）

### 🎨 カラースキーム
5種類のカラーパターン：
- 🌈 レインボー（グラデーション）
- 🔥 ファイア（暖色系）
- 🌊 オーシャン（寒色系）
- 💜 ネオン（紫系）
- ⚪ モノクロ

### 🖼️ 背景画像
- JPG, PNG, GIF, WebP 形式に対応
- 透明度調整（10%〜100%）
- ぼかし効果（0px〜20px）

### 🎚️ 周波数設定
- 下限周波数: 20Hz〜8,000Hz
- 上限周波数: 500Hz〜22,050Hz
- プリセット:
  - 低音域（20Hz〜500Hz）
  - 中音域（500Hz〜4,000Hz）
  - 高音域（4,000Hz〜20,000Hz）
  - 人の声（80Hz〜3,000Hz）
  - 全帯域（20Hz〜16,000Hz）

### 🔴 録画機能
- スペクトラム表示を動画として録画
- 音声付きWebM形式で出力
- 再生終了時に自動ダウンロード

## 🚀 使い方

### 基本操作

1. **音声ファイルを選択**
   - 左側のエリアに音声ファイルをドラッグ＆ドロップ
   - またはクリックしてファイルを選択

2. **再生**
   - 「▶ 再生」ボタンをクリック

3. **カスタマイズ**
   - スタイル、カラー、感度、周波数範囲を調整

### 背景画像を設定

1. 右側のエリアに画像ファイルをドラッグ＆ドロップ
2. 透明度とぼかしをスライダーで調整
3. 「削除」ボタンで背景を削除

### 動画として録画

1. 音声ファイルを読み込む
2. 各種設定を調整
3. 「🔴 録画」ボタンをクリック
4. 再生が終了すると自動的にWebMファイルがダウンロード

## 🛠️ 技術仕様

### 使用技術
- **Web Audio API**: 音声解析（FFT）
- **Canvas API**: リアルタイム描画
- **MediaRecorder API**: 動画録画
- **React 18**: UIフレームワーク
- **Tailwind CSS**: スタイリング

### オーディオ解析
- FFTサイズ: 2048
- サンプルレート: 44,100Hz（デバイス依存）
- 周波数ビン数: 1024
- フレームレート: 60fps

### 出力仕様
- キャンバス解像度: 1920×1080（16:9）
- 録画形式: WebM（VP9/VP8 + Opus）
- ビットレート: 8Mbps

## 🌐 デプロイ方法

### GitHub Pages

1. GitHubでリポジトリを作成
2. `index.html` と `README.md` をアップロード
3. Settings → Pages → Source を `main` ブランチに設定
4. 公開URLにアクセス

### その他の静的ホスティング
- Netlify
- Vercel
- Firebase Hosting
- Amazon S3

`index.html` を配置するだけで動作します。

## ⚠️ 注意事項

- **ブラウザ対応**: Chrome, Firefox, Edge, Safari（最新版推奨）
- **録画機能**: 一部のブラウザでは対応コーデックが異なる場合があります
- **パフォーマンス**: 高解像度での録画はCPU/GPUに負荷がかかります
- **音声ファイル**: ローカルファイルのみ対応（URL読み込み非対応）
